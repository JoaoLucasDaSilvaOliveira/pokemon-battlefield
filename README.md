
# POKEMON BATTLEFIELD

Este é um projeto para as matérias de Desenvolvimento Web II e Programação Orientada à Objetos II.

Trata-se de um sistema de batalha de pokemons entre usuários, cada um com o seu treinador e seus pokemons.





## Autores

- [@JoaoLucasDaSilvaOliveira](https://www.github.com/JoaoLucasDaSilvaOliveira)
- [@tifascode](https://www.github.com/tifascode)

# Funcionalidades

O projeto baseia-se em três grandes blocos:

- API RESTful para realizar ações (CRUD e Batalhas)
- Banco de dados relacional
- Paginas reativas (SPA) para UX-UI


## Stack utilizada

**Front-end:** React - Vite

**Back-end:** Spring - Starters: Web, JPA, Validation; Lombok

**Database:** H2 Database

**Helpers:** MapStruct: mapeador de classes.
